iWin Web XP - web emulator (Windows XP look) for iPhone.

How to use:
1) Upload these files to a free host (glitch.com, replit.com, or any static host).
   - On Glitch: New Project -> Import from Zip -> upload iWinWebXP.zip
   - On Replit: Create new Repl -> Import from Zip
2) Open the project URL in Safari on your iPhone.
3) Use Share -> Add to Home Screen to pin it like an app (PWA).

Notes:
- Files and drawings are stored in localStorage of the browser (on-device).
- Long-press the green Start button to switch language (UK <-> RU).
- This is fully free and works without Apple Developer account.
